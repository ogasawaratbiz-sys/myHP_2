# 中小企業診断士 個人事務所 HP

中小企業診断士の個人事務所ウェブサイト。補助金支援・マーケティング支援を主なサービスとして展開。

## 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **スタイリング**: Tailwind CSS v4
- **言語**: TypeScript
- **ホスティング**: Vercel（推奨）

## ページ構成

| ページ | パス | 内容 |
|--------|------|------|
| トップ | `/` | ヒーロー・サービス概要・選ばれる理由・FAQ |
| 補助金支援 | `/services/subsidy` | 補助金一覧・支援の流れ |
| マーケティング支援 | `/services/marketing` | 支援メニュー・支援の流れ |
| 事務所概要 | `/about` | 代表プロフィール・経歴・事務所情報 |
| お問い合わせ | `/contact` | お問い合わせフォーム |

## ローカル開発

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) でサイトを確認できます。

## カスタマイズが必要な箇所

`〇〇` となっている箇所を実際の情報に置き換えてください：

- 事務所名・代表者名
- 住所・電話番号・メールアドレス
- 代表者プロフィール・経歴
- プロフィール写真（`public/` ディレクトリに配置）
